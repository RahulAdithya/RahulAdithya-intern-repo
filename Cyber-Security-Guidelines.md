### 1. Common Cyber Security Threats in Remote Work
* **Phishing:** Deceptive emails designed to steal credentials or install malware.
* **Insecure Wi-Fi:** Working from public networks without a VPN can lead to data interception.
* **Weak Authentication:** Using simple or reused passwords across different work platforms.
* **Social Engineering:** Manipulating individuals into divesting confidential information.

### 2. Best Practices for Device & Account Security
* **Keep Software Updated:** Regularly patch your OS and applications to fix vulnerabilities.
* **Disk Encryption:** Ensure Mac FileVault is active to protect data if the device is lost.
* **Minimalist Access:** Only install necessary software and use standard (non-admin) accounts for daily tasks.

### 3. Importance of Locking Your Computer
Locking your computer prevents unauthorized physical access to sensitive company data, GitHub repositories, and internal communications while you are away from your desk. It is a critical habit for maintaining data integrity in shared or remote spaces.

### 4. Handling Phishing & Suspicious Links
* **Verify the Sender:** Carefully inspect the sender's email address for slight misspellings.
* **Hover Before Clicking:** Hover over links to see the actual destination URL.
* **Report:** If a link looks suspicious, do not click it; report it to the security team or your supervisor immediately.

### 5. Strong Passwords & Password Managers
* **Strong Passwords:** Should be long (12+ characters), unique for every account, and include a mix of symbols, numbers, and letters.
* **Password Managers:** These tools store encrypted credentials, allowing you to use high-complexity passwords without needing to memorize them, which prevents credential stuffing attacks.

### 6. Two-Factor Authentication (2FA)
2FA provides a vital second layer of security. Even if a password is compromised, an attacker cannot gain access without the second factor (like a code from an authenticator app). It should be enabled on all work accounts, especially GitHub and Gmail, immediately.
